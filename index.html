How I Built an AI-Powered News Intelligence API for Real Companies

A high school junior's journey from idea to production-ready solution

By Keshav Shrivastava | Summer 2024

As a junior in high school, I never imagined that a summer project would end up being integrated into a real company's Microsoft 365 Copilot system. Yet here I am, writing about NewLens API — an AI-powered solution that's helping sales and marketing teams stay on top of company news without the manual hassle.

The Problem That Started It All

Picture this: You're on a sales team tracking 50+ potential clients, or you're in marketing monitoring competitors. Every morning, you'd have to:

  • Visit multiple news websites
  • Search for each company individually
  • Sift through irrelevant articles
  • Manually summarize key information
  • Repeat this process daily

This tedious routine wastes hours that could be spent on actual selling or strategizing. I realized there had to be a better way.

What is NewLens API?

NewLens API is an intelligent news aggregation service that:

  • Automatically finds the latest news about specific companies
  • Uses AI to summarize content into concise, actionable insights
  • Filters by categories like press releases, financial news, and headlines
  • Integrates seamlessly with existing tools and workflows

Think of it as your personal news assistant that never sleeps and never gets tired of searching for company updates.

The Technical Journey

Choosing the Right Architecture

After evaluating multiple deployment options including Google Cloud Run, Azure Container Instances, Railway, and traditional server hosting, I chose AWS Lambda for several key reasons:

Cost-Effective for Students: With Lambda's pay-per-execution model, I only pay for actual usage. For a student budget, this was crucial — no $20/month hosting fees for an app that runs occasionally.

Perfect for the Use Case: News intelligence doesn't need 24/7 server uptime. Lambda's event-driven architecture aligns perfectly with periodic analysis tasks.

Professional Scalability: If this project grows, Lambda scales automatically without requiring architectural rewrites.

Why Not the Other Options?

Railway/Render: These platforms looked cool and were beginner-friendly, but they cost $5–20/month even when my app wasn't running. That's too steep for a student budget when Lambda could handle the same workload essentially for free.

Google Cloud Run: Actually quite similar to Lambda with comparable pricing, but AWS had better documentation and more learning resources for someone just starting with cloud development.

Traditional Web Hosting: Would have cost $10+ monthly and required learning server management — skills that weren't really relevant to my core project goals. Plus, I'd be paying for resources I wasn't using 99% of the time.

The decision ultimately came down to focusing on what mattered: building AI-powered features rather than managing infrastructure. AWS Lambda let me put "serverless architecture" on my resume while keeping costs near zero!

The Tech Stack

  • Python for the core Lambda function
  • Perplexity AI API for intelligent news analysis and summarization
  • AWS Lambda for serverless deployment
  • API Gateway for REST endpoint management
  • CloudWatch for monitoring and logging

API Design Philosophy

I designed the API to be simple yet powerful. Here's how it works:

{
  "ismock": false,
  "clients": [
    {
      "name": "RLDatix Life Sciences",
      "url": "https://www.rldatixlifesciences.com/"
    }
  ],
  "categories": [
    "Latest Headlines",
    "Press Release",
    "Financial News"
  ]
}

The response delivers structured, AI-summarized insights that can be immediately consumed by other systems or presented to users.

Real-World Impact: Microsoft 365 Copilot Integration

The most exciting validation came when a real company integrated NewLens API into their Microsoft 365 Copilot system. Their sales and marketing teams can create an Excel document in their SharePoint portal and create a simple Office 365 Copilot Agent to call NewsLens API.

They can ask "Show me the latest news" or simply ask for a specific company: "Show me the latest news for RLDatix LifeSciences"

And instantly receive:

Press Release Updates

  • MediSpend and RLDatix Life Sciences Merge — Creating a unified SaaS platform for regulatory compliance across 300+ global pharmaceutical, biotech, and medical device companies
  • Date: September 2, 2025

Latest Headlines

  • RLDatix Expands Leadership Team — Appointed key executives including Chief People Officer and Senior VP of Global Operations to drive growth and operational excellence
  • Date: July 21, 2025

This integration saves their team hours of manual research weekly, allowing them to focus on relationship building and strategic planning.

Key Lessons Learned

1. Start With Real Problems

The best projects solve genuine pain points. By focusing on the tedious news-monitoring workflow, I created something people actually wanted to use.

2. Architecture Decisions Matter

Choosing AWS Lambda wasn't just about cost — it was about aligning technology choices with actual usage patterns and future scalability needs.

3. AI APIs Are Powerful Building Blocks

Rather than building AI from scratch, leveraging Perplexity's API allowed me to focus on the core value proposition while delivering professional-quality results.

4. Security and API Design

Requiring users to provide their own Perplexity API keys ensures security and compliance while keeping operational costs manageable.

What's Next?

The positive response has been overwhelming. Future enhancements I'm considering:

  • Sentiment analysis for deeper insights
  • Historical trend tracking to identify patterns
  • Webhook notifications for real-time updates
  • Custom categorization for industry-specific needs

For Fellow Students and Developers

If you're inspired to build something similar, here are my recommendations:

Start Small: Begin with a focused use case rather than trying to solve everything.

Choose Modern Tools: Serverless architectures like AWS Lambda reduce operational complexity and costs.

Think Integration: Design APIs that play well with existing tools (like Microsoft 365, Slack, or custom dashboards).

Solve Real Problems: The best validation is when real companies integrate your solution into their workflows.
